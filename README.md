# 22 Strong Foundation

Astro site for the 22 Strong Foundation nonprofit.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Testing

```bash
npm test
```

## Deployment

Static output in `dist/`. Deploy anywhere (Netlify, Vercel, Cloudflare Pages, etc.)

## Content

Edit `src/content/home.json` for copy, `src/content/site.json` for config.
